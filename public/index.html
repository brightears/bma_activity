BMA Activity Report System

API Status: Active

Welcome to BMA Activity Report API

This is the backend API for managing BMA activities and reports.

Version

1.0.0

Environment

Development

Database

PostgreSQL

Available Endpoints

GET/health - Health check
GET/api/activities - Get all activities
POST/api/activities - Create new activity
GET/api/activities/:id - Get activity by ID
PUT/api/activities/:id - Update activity
DELETE/api/activities/:id - Delete activity

For API documentation and usage, please refer to the project documentation.
